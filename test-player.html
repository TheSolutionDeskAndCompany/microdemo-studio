How to Use

This test page is loading a demo with ID: vvxi7n6z

Prerequisites:

  1. Studio server running on http://localhost:3002
  2. Player dev server running on http://localhost:5173

To test with a new demo:

curl -X POST http://localhost:3002/api/demos \
  -H "content-type: application/json" \
  -d '{"title":"New Demo","steps":[{"index":0,"action":"click","selector":"#example","caption":"Example click"}]}'

Then update the data-id attribute in the microdemo-player element.
